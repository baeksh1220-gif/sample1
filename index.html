AI에게 의도를,
코드는 AI가

2025년 안드레이 카파시가 제시한 개념. 개발자는 세세한 구현보다 무엇을 만들고 싶은지에 집중하고, AI가 실제 코드를 생성·수정합니다. 실수가 나면? 그냥 AI에게 고쳐달라고 하면 됩니다.



"완전히 진심으로 코드를 잊어버려라.
존재하는 것은 당신이 원하는 것뿐이다."
— Andrej Karpathy

02
⚡

AI가 구현

Claude, Cursor, Copilot 같은 AI가 즉시 코드를 생성합니다. 실행하고 결과를 확인하세요. 틀려도 괜찮습니다.

03
🔄

반복과 개선

마음에 안 들면 다시 말하면 됩니다. "색상 바꿔줘", "더 빠르게 만들어줘". 당신의 감각이 기준입니다.

// 왜 바이브 코딩인가

개발의 재정의

🚀

극단적 속도

아이디어에서 프로토타입까지 몇 시간이 아닌 몇 분. 반복 속도가 빨라지면 더 많이 시도하고 더 빠르게 배웁니다.

🧠

창의성 최우선

문법과 보일러플레이트에 쏟던 인지 자원을 문제 해결과 UX 설계에 집중할 수 있습니다.

🌍

접근성 혁명

개발 경험이 없어도 됩니다. 디자이너, 기획자, 도메인 전문가 누구나 자신의 아이디어를 직접 구현할 수 있습니다.

🎯

VIBE = 의도 + 맥락 + 감각

단순히 "AI한테 코드 시키기"가 아닙니다. 분위기(vibe)를 전달하고 AI와 협업하며 다듬어가는 새로운 창작 과정. 당신은 디렉터, AI는 개발팀입니다.
